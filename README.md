# AI-for-Bharat-Exam-Sarthi
Exam-Sarthi is an AI-powered agentic study companion designed to eliminate "syllabus paralysis" for students in Bharat. By using Retrieval-Augmented Generation (RAG) and AWS Bedrock, it analyzes syllabus PDFs and Previous Year Questions (PYQs) to identify high-weightage topics and generate a personalized, hour-by-hour "3-Day Crash Course" schedule.
